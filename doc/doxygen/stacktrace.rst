.. _stacktrace_h:

stacktrace.h
============

This header defines a utility for generating stack trace if the program crashes.
It can be enabled by defining ``SCTL_SIG_HANDLER`` in code or using the compiler flag ``-DSCTL_SIG_HANDLER``.
